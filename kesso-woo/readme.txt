A collection of WooCommerce utilities and enhancements to improve your store's functionality.

== Description ==

Kesso WooCommerce provides a suite of utilities and enhancements for WooCommerce stores, helping you manage and optimize your online store more effectively.

== Features ==

* Product synchronization across Polylang translations
* Additional features coming soon
* Simple enable/disable toggles for each feature
* Clean, modern admin interface

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/kesso-woo` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Ensure WooCommerce is installed and activated
4. Go to # WooCommerce in the admin menu to configure features

== Frequently Asked Questions ==

= Does this require WooCommerce? =

Yes, this plugin requires WooCommerce to be installed and activated.

= What features are available? =

Currently, the plugin includes product synchronization for Polylang. More features will be added in future updates.
