Self-hosted accessibility widget for WordPress with font resizing, contrast modes, and keyboard navigation helpers.

== Description ==

A minimal, self-hosted WordPress accessibility plugin that provides a frontend widget for visitors to customize their browsing experience. No external dependencies, no SaaS accounts, no CDN scripts.

== Features ==

* Font size controls (increase/decrease)
* Contrast modes (grayscale, high contrast, negative, light background)
* Underline links toggle
* Readable font toggle
* Pause animations
* Keyboard accessibility helpers
* Skip to content link
* Focus outline customization
* LocalStorage persistence
* WordPress Customizer integration

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/accessibility-widget` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Accessibility > Settings to configure the widget
4. Customize appearance in Appearance > Customize > Accessibility

== Frequently Asked Questions ==

= What features does this plugin provide? =

The plugin provides a frontend accessibility toolbar with font resizing, contrast modes, link underlining, readable font, pause animations, and keyboard navigation helpers.

= Does this plugin require an account? =

No. This is a completely self-hosted plugin with no external dependencies.

= Can I customize the widget appearance? =

Yes. You can customize colors, position, and icon through the WordPress Customizer under Appearance > Customize > Accessibility.

= Does this plugin make my site fully accessible? =

This plugin provides tools to help visitors customize their experience, but it does not automatically make your site fully accessible. You should still follow WCAG guidelines and conduct accessibility audits.
