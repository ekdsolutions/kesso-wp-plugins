== Description ==

Kesso Messages is a WordPress plugin that centralizes all admin notifications and warnings into a clean, organized notification popup accessible via a bell icon in the WordPress admin bar.

Instead of having admin notices scattered across the top of every admin page, this plugin collects them and displays them in a single, easy-to-manage popup interface.

== Features ==

* Notification bell icon in WordPress admin bar
* Centralized notification popup
* Settings page to control which message types appear in popup vs. regular display
* Automatic collection of all WordPress admin notices
* Badge counter showing number of unread notifications
* Clean, modern UI matching Kesso plugin design standards

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/kesso-msg` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure settings in the Messages menu item
