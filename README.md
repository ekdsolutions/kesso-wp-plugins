# Kesso WordPress Plugins

A collection of WordPress plugins developed by [kesso.io](https://kesso.io).

## Plugins

### kesso-access
Accessibility widget plugin that provides tools to improve website accessibility for all users.

### kesso-cookies
GDPR-compliant cookie consent banner with script blocking and category management.

### kesso-init
WordPress initialization wizard for setting up plugins and themes.

### kesso-msg
Centralized admin notifications system that displays WordPress admin messages in a minimizable popup.

### kesso-woo
WooCommerce utilities and enhancements, including Polylang synchronization for WooCommerce products.

## Installation

Each plugin can be installed independently by copying the plugin folder to your WordPress `wp-content/plugins/` directory.
